# Amazon Clone

A responsive Amazon-inspired e-commerce web application developed using HTML, CSS, and JavaScript.

## Features

- Amazon-style homepage with product listings
- Fully functional shopping cart system
- Add products to cart and view selected items
- Place orders directly from the cart page
- Dedicated Orders page to view all placed orders
- Package tracking functionality for ordered products
- Track Package option with shipment progress details
- View All Orders feature available across pages
- Clickable Amazon logo that redirects users to the homepage
- Responsive design for mobile and desktop devices
- User-friendly navigation and interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- Home Page
- Cart Page
- Orders Page
- Package Tracking Page

## Purpose

This project was developed to strengthen frontend web development skills by building a multi-page e-commerce application with cart management, order handling, navigation, and package tracking features.

## Learning Outcomes

- DOM Manipulation
- Event Handling
- JavaScript Logic Building
- Multi-page Navigation
- Responsive Web Design
- E-commerce Workflow Implementation
